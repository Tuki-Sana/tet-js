# PWA アイコン

`icon-192.png`（192×192）と `icon-512.png`（512×512）をこのフォルダに置くと、ホーム画面に追加時のアイコンとして使われます。

## アイコンの生成

1. ブラウザで `generate-icons.html` を開く
2. 「192px を保存」「512px を保存」をクリックして PNG をダウンロード
3. ダウンロードしたファイルをこのフォルダに `icon-192.png` / `icon-512.png` として保存

アイコンを用意しない場合も PWA は動作しますが、インストール時にデフォルトのアイコンが使われます。
